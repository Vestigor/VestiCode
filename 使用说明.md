# 使用说明

VestiCode 发布版为**自包含单文件**：目标机**无需安装 .NET 运行时、无需联网**，下载解压后即可安装使用。

## 1. 下载

前往 [Releases](https://github.com/Vestigor/VestiCode/releases/tag/VestiCode_v0.1.0)，按平台下载对应安装包：

| 平台 | 安装包 |
|------|--------|
| macOS (Apple Silicon) | `vesticode-0.1.0-osx-arm64.tar.gz` |
| Windows x64 | `vesticode-0.1.0-win-x64.zip` |
| Linux x64 | `vesticode-0.1.0-linux-x64.tar.gz` |

## 2. 解压

- macOS / Linux：`tar -xzf vesticode-0.1.0-<平台>.tar.gz`
- Windows：右键解压 `.zip`

## 3. 安装

进入解压后的目录，查看其中的 `README.md`（分平台安装说明），按它执行安装脚本：

- macOS / Linux：`./install.sh`
- Windows（PowerShell）：`.\install.ps1`

## 4. 验证

```bash
vesticode --version
```

## 5. 首次使用

首次运行进入交互式配置向导（选择 LLM 提供方 / 模型 / 填 API Key），配置写入 `~/.vesticode/appsettings.json`。

更多用法见 [使用教程 docs/TUTORIAL.md](docs/TUTORIAL.md)。
